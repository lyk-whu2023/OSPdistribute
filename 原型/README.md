# OSP商城前端原型

这是一个基于HTML、CSS和JavaScript的分布式在线购物平台前端原型，用于展示平台的主要功能和界面设计。

## 项目结构

```
原型/
├── css/
│   └── style.css          # 全局样式文件
├── js/
│   └── script.js          # 全局脚本文件
├── index.html             # 首页
├── products.html          # 商品列表页
├── cart.html              # 购物车页
├── stores.html            # 店铺列表页
├── blog.html              # 博客列表页
├── user.html              # 个人中心页
└── README.md              # 项目说明文件
```

## 功能说明

### 首页
- 轮播图展示
- 商品分类推荐
- 热销商品展示
- 秒杀活动展示
- 推荐店铺展示
- 最新博客展示

### 商品列表页
- 商品分类筛选
- 价格区间筛选
- 品牌筛选
- 商品排序（综合、销量、价格）
- 商品列表展示
- 分页功能

### 购物车页
- 商品列表展示
- 商品数量调整
- 商品删除
- 全选/取消全选
- 购物车总计
- 去结算功能

### 店铺列表页
- 店铺类型筛选
- 商品分类筛选
- 评分筛选
- 店铺排序（综合、关注数、评分）
- 店铺列表展示
- 店铺关注功能
- 分页功能

### 博客列表页
- 博客分类导航
- 标签云
- 最新文章推荐
- 博客排序（最新、热门、推荐）
- 博客列表展示
- 分页功能

### 个人中心页
- 个人信息展示
- 订单管理（全部、待支付、待发货、待收货、已完成、已取消）
- 地址管理（添加、编辑、删除、设为默认）
- 收藏管理（查看、加入购物车、取消收藏）
- 账户设置（修改密码、绑定手机）

## 技术实现

- **HTML5**：页面结构
- **CSS3**：页面样式，包括响应式设计
- **JavaScript**：页面交互功能
- **Font Awesome**：图标使用（通过CDN引入）
- **Trae API**：生成模拟图片

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件即可访问首页
2. 通过导航栏可以跳转到其他页面
3. 所有页面都包含了模拟数据，可以直接查看和交互

## 注意事项

- 这只是一个前端原型，不包含后端功能
- 所有数据都是模拟的，仅用于展示界面和交互
- 响应式设计支持不同屏幕尺寸
- 部分功能使用了JavaScript实现交互效果
