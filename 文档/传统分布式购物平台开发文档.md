# 传统分布式购物平台开发文档

## 1. 项目概述

本项目是一个基于传统分布式架构的在线购物平台，提供商品浏览、购物车管理、订单处理、用户管理等核心功能，同时为管理员提供完善的后台管理系统。项目采用垂直分层架构，按业务域整合功能模块，简化部署和维护流程，适合中小规模应用场景。

### 7.1 JWT 认证

```java
// JWT工具类
@Component
public class JwtUtil {
    private String secret = "your-secret-key";
    private long expiration = 3600000; // 1小时
    
    public String createToken(Long userId, String username, String role) {
        Map<String, Object> claims = new HashMap<>();
        claims.put("userId", userId);
        claims.put("username", username);
        claims.put("role", role);
        
        return Jwts.builder()
            .setClaims(claims)
            .setExpiration(new Date(System.currentTimeMillis() + expiration))
            .signWith(SignatureAlgorithm.HS512, secret)
            .compact();
    }
    
    public Claims parseToken(String token) {
        return Jwts.parser()
            .setSigningKey(secret)
            .parseClaimsJws(token)
            .getBody();
    }
}
```

### 7.4 商品秒杀实现

```java
// 秒杀服务
@Service
public class SeckillService {
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    @Autowired
    private SeckillProductMapper seckillProductMapper;
    @Autowired
    private RabbitTemplate rabbitTemplate;
    
    // 初始化秒杀商品库存到Redis
    @PostConstruct
    public void initSeckillStock() {
        List<SeckillProduct> products = seckillProductMapper.selectList(null);
        for (SeckillProduct product : products) {
            redisTemplate.opsForValue().set("seckill:stock:" + product.getId(), product.getStock());
        }
    }
    
    // 秒杀抢购
    public boolean seckill(Long productId, Long userId) {
        // 检查是否已经抢购过
        if (redisTemplate.hasKey("seckill:order:" + userId + ":" + productId)) {
            return false;
        }
        
        // 减库存
        Long stock = redisTemplate.opsForValue().decrement("seckill:stock:" + productId);
        if (stock < 0) {
            // 库存不足，回滚
            redisTemplate.opsForValue().increment("seckill:stock:" + productId);
            return false;
        }
        
        // 标记用户已抢购
        redisTemplate.opsForValue().set("seckill:order:" + userId + ":" + productId, "1", 24, TimeUnit.HOURS);
        
        // 发送消息到队列，异步处理订单
        SeckillMessage message = new SeckillMessage();
        message.setProductId(productId);
        message.setUserId(userId);
        rabbitTemplate.convertAndSend("seckill-exchange", "seckill.order", message);
        
        return true;
    }
    
    // 异步处理秒杀订单
    @RabbitListener(queues = "seckill-order-queue")
    public void handleSeckillOrder(SeckillMessage message) {
        // 创建秒杀订单
```

### 7.5 统计功能实现

```java
// 统计服务
@Service
public class StatsService {
    @Autowired
    private OrderMapper orderMapper;
    @Autowired
    private UserMapper userMapper;
    @Autowired
    private ProductMapper productMapper;
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    // 销售统计
    public SalesStatsDTO getSalesStats(Date startDate, Date endDate) {
        // 从数据库查询
        List<Order> orders = orderMapper.selectList(
            new QueryWrapper<Order>()
                .between("create_time", startDate, endDate)
                .eq("status", "completed")
        );
        
        // 计算销售额
        BigDecimal totalSales = orders.stream()
            .map(Order::getAmount)
            .reduce(BigDecimal.ZERO, BigDecimal::add);
        
        // 计算订单数
        int orderCount = orders.size();
        
        SalesStatsDTO stats = new SalesStatsDTO();
        stats.setTotalSales(totalSales);
        stats.setOrderCount(orderCount);
        stats.setStartDate(startDate);
        stats.setEndDate(endDate);
        
        return stats;
    }
    
    // 实时统计（使用Redis）
    public RealTimeStatsDTO getRealTimeStats() {
        // 从Redis获取实时数据
        Long seckillCount = redisTemplate.opsForValue().get("stats:seckill:count") != null 
            ? (Long) redisTemplate.opsForValue().get("stats:seckill:count") 
            : 0;
        
        Long blogViewCount = redisTemplate.opsForValue().get("stats:blog:view:count") != null 
            ? (Long) redisTemplate.opsForValue().get("stats:blog:view:count") 
            : 0;
        
        RealTimeStatsDTO stats = new RealTimeStatsDTO();
        stats.setSeckillCount(seckillCount);
        stats.setBlogViewCount(blogViewCount);
        stats.setTimestamp(new Date());
        
        return stats;
    }
}
```

本项目采用传统分布式架构，通过垂直分层和模块整合，降低了系统复杂度和维护成本，同时保留了核心功能和性能特性。项目使用了现代化的技术栈，包括 Spring Boot、Vue 3、Redis、RabbitMQ 等，为业务的持续发展提供了有力的技术支撑。

### 功能亮点

1. **短信登录**：使用阿里云短信服务，实现了便捷的短信验证码登录功能，提升了用户体验。

2. **博客系统**：支持博客的创建、编辑、搜索和管理，为平台增加了内容营销能力。

3. **商品秒杀**：使用 Redis 实现高并发秒杀，结合消息队列异步处理订单，确保系统在高并发场景下的稳定性。

4. **店铺关注**：实现了用户对店铺的关注功能，增强了用户与店铺的互动。

5. **统计功能**：提供了全面的统计分析功能，包括销售统计、用户统计、商品统计和实时统计，为运营决策提供数据支持。

### 技术亮点

1. **传统分布式架构**：通过垂直分层和模块整合，简化了系统架构，降低了部署和维护复杂度。

2. **高性能处理**：通过 Redis 缓存、消息队列等技术，提高了系统的并发处理能力。

3. **实时数据**：使用 Redis 存储实时统计数据，提供实时的业务 insights。

4. **搜索引擎**：集成 Elasticsearch，提供高效的商品和博客搜索功能。

5. **安全性**：实现了完善的认证授权机制和数据安全措施，保障系统安全。

通过本项目的开发，我们实现了完整的在线购物功能，同时积累了传统分布式系统开发的经验，为未来的项目开发奠定了基础。该平台具有良好的可扩展性和可维护性，可以根据业务需求不断添加新功能和优化性能。
